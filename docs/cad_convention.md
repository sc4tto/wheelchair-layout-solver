# CAD convention

Recommended layers:

- `ACC_ROOM`
- `ACC_WALL`
- `ACC_OBSTACLE`
- `ACC_DOOR`
- `ACC_WC`
- `ACC_SINK`
- `ACC_SHOWER`
- `ACC_START`
- `ACC_TARGET`
- `ACC_FUNCTIONAL_AREA`

Recommended block attributes:

- `ID`
- `TYPE`
- `MOVABLE`
- `X_TOL_MINUS`
- `X_TOL_PLUS`
- `Y_TOL_MINUS`
- `Y_TOL_PLUS`
- `ROTATION_VALUES`
- `TRANSFER_SIDE`

Use metres internally. Importers must explicitly convert source units.
